# tinyShare-multiThread

#### 功能简介:

1. P2P传输,支持文件夹整个传输,利用线程池,每个文件都是一个单独的线程(所以如果你有很多很多小文件,请将其打包后发送,否则会很慢)
3. 支持文件断点续传,接收方接受的部分请不要修改其内容,也不要更改其位置,否则会重新传输整个文件


环境:
QT6.8 MINGW64 QTcreator14.0.2(community)
